Introducción

What is HTML?

HTML es un lenguaje de etiquetado diseñado con el proposito de crear páginas web, este se utiliza para generar la estructura de una pagina mediante etiquetas que representan elementos, es el lenguaje estandar que entienden los navegadores web

Explain the head and the body tags

La etiqueta de "head" es necesaria en todos los documentos html, se utiliza para agregar metadatos a la pagina, como el lenguaje, el titulo, el metodo de encoding y otros elementos mas necesarios para la visualización de la página en cualquier navegador

Basic tags

Explica las etiquestas: section, article, aside, header y footer

Cada una de estas etiquetas se utilizan para dar estructura a la pagina, sirviendo como organizadores de la informacion. Cada una se encarga de lo siguiente:

  • Section: Representa una sección de la pagina, tiene un proposito general
  • Article: Se utiliza para envolver contenido que es en una parte independiente, practicamente como un articulo en un periodico, su uso más común es en blogs o sitios de noticias
  • Aside: Se utiliza normalmente para agregar información secundaria en una pagina, su uso más común es como barra lateral mostrando publicidad.
  • Header: Es la primera parte que se observa de una página, normalmente incluye la barra de navegación, algunas veces el Hero se inserta aqui, practicamente es el encabezado de una página web
  • Footer: Suele representar la ultima parte de una página web y es muy comun en la mayoría de páginas, es la sección hasta abajo de las páginas y normalmente tiene enlaces a redes sociales, medios de comunicación o simplemente el logo de la página

Introducción

What is CSS?

Es un lenguaje de diseño utilizado justamente para definir la apariencia visual en documentos html, trabaja conjuntamente con HTML permitiendo separar los estilos en un archivo separado, utilizado selectores como clases, ids o etiquetas para asignar estilos y propiedades a cada elemento de una página.

What's the purpouse of CSS?

El principal proposito de CSS es definir los estilos y apariencia visual de una página web

How do you apply a CSS to an HTML

  • Mediante un archivo externo, utilizando: <link rel="stylesheet" href="style.css" /> dentro de la etiqueta de head
  • Internamente dentro de la etiqueta head tambien se puede definir una etiqueta style donde se puede escribir el codigo css dentro del mismo documento html
  • Inline dentro de cada elemento, tambien es posible incrustar codigo css directamente dentro de cada etiqueta individualmente mediante la propiedad "style"

What is a selector

Un selector es un identificador que se utiliza para elegir entre elementos de un documento HTML, hay de diferentes tipos, por ejemplo clases, ids, etiquetas o incluso es posible seleccionar todo.

What is a CSS rule

Una regla de CSS es un código que establece como se deberia ver un elemento, cada regla requiere de seleccionar un elemento ademas de justamente las caracteristicas que se le quieren asignar a dicho elemento.

Basic CSS

Explain a:

  • Tag selector: Selecciona todos los elementos que tengan el nombre de una etiqueta específica (como p, h1, div) y les aplica el estilo de forma global.
  • Class selector: Selecciona elementos que tienen un atributo class específico. Se identifica con un punto (.) antes del nombre.
  • Tag and class selector: Es cuando combinas ambos para ser más específico, por ejemplo, p.rojo solo afectará a los párrafos que tengan la clase "rojo", ignorando otros elementos con esa misma clase.
  • Id selector: Selecciona un único elemento que tenga un atributo id específico. Se identifica con un gato (#) y tiene más prioridad que las clases.

What's a class?

Es un atributo que puedes repetir en varios elementos de tu HTML. Sirve para agrupar diferentes etiquetas que quieres que compartan el mismo diseño

What's an id?

Es un identificador único para un solo elemento en toda la página. Se usa ademas de para establecer estilos para identificar un elemento único para darle interactividad

How are they related to CSS?

Basicamente son la conexión que usa CSS para saber a que parte del HTML debe modificar. Sin ellos, no podríamos diferenciar un párrafo de otro al dar estilos.

Explain the CSS Box Model

El modelo Box es la forma en la que los objetos se comportan en CSS, ya que este lenguaje cuenta con 4 propiedades clave que cada contenedor tiene, estas son: margin(distancia a otros elementos), padding (espacio extra que abarca el elemento), content (el contenido de dicha caja) y border. Estos elementos conforman el layout de cada elemento como un div, section, etc...

Explain the Media Queries

Los Media Queries son reglas que se pueden establecer en CSS con el fin de crear paginas responsivas, practicamente funcionan similar a condicionales, ya que te permite definir estilos condicionales segun la orientacion del dispositivo, el tamaño de la pantalla y otras caracteristicas.
